# AnnotateKit Connector Privacy Policy

Effective: July 22, 2026

AnnotateKit is a developer tool for capturing visual feedback inside Debug builds of iOS applications. The self-hosted connector receives only data that a developer deliberately sends from the AnnotateKit overlay: annotation text, UI/accessibility context, app and device context, workflow status, and optional discussion replies.

The open-source connector stores sessions in process memory and does not create analytics, advertising profiles, or sell personal data. Data is sent to the server endpoint selected by the user and is visible to MCP clients that the user connects to that endpoint. Restarting the connector clears its in-memory data. Screenshots remain local unless a user separately transfers them.

Operators who deploy the connector are responsible for their hosting provider, access controls, retention, and applicable law. Do not expose an unauthenticated connector containing sensitive feedback to the public internet. AnnotateKit is intended for development and testing, not production end-user data.
